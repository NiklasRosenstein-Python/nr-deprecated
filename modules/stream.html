stream

Provides a stream class that wraps an iterator and supports a number of methods to further wrap and process it. Many methods can be used as both instance and static methods.

Example

from nr.stream import stream

for friend in stream(persons).attr('iter_friends').call().concat().unique():
  print(friend)
